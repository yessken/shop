# Тестовое задание для Click Intellect
**Автор:** Ескендир Толеубеков

## Описание проекта

Приложение реализует функционал **разделения списка товаров на группы**, при этом:

- Общая цена товаров в группе **не превышает 200 евро**.
- Общая цена каждой группы **максимально приближена к лимиту 200 евро**.

Реализованы два основных эндпоинта:

1. `/upload` — загрузка товаров из Excel-файла (`.xlsx`) и сохранение в базу данных.
2. `/group-products` — генерация групп товаров на основе загруженных данных.

Проект разработан на **ASP.NET Core 7**, с использованием **Entity Framework Core** и **Swagger UI**.

---

## Структура проекта

- **Data/** — модели данных: `Product`, `ProductGroup`, `ProductGroupItem`.
- **Services/** — сервис для логики формирования групп товаров (`ProductGroupingService`).
- **Program.cs** — настройка endpoints, DI и Swagger.

---

## Работа с приложением

### Загрузка Excel

Отправьте POST-запрос на `/upload` с файлом `.xlsx`.

Формат файла:

| Наименование        | Единица измерения | Цена за единицу, евро | Количество, шт. |
|--------------------|-----------------|---------------------|----------------|
| Ручка              | штука           | 1.5                 | 150            |
| Бумага А4          | упаковка        | 2.6                 | 50             |
| Доска маркерная    | штука           | 32                  | 11             |
| Блокнот 100 страниц| штука           | 10                  | 5              |

### Генерация групп

Отправьте GET-запрос на `/group-products`.

Пример ответа:

```json
[
  {
    "GroupName": "Группа 1",
    "Items": [
      { "Name": "Доска маркерная", "UnitName": "штука", "Price": 32, "Quantity": 6 },
      { "Name": "Бумага А4", "UnitName": "упаковка", "Price": 2.6, "Quantity": 3 }
    ]
  },
  {
    "GroupName": "Группа 2",
    "Items": [
      { "Name": "Доска маркерная", "UnitName": "штука", "Price": 32, "Quantity": 5 },
      { "Name": "Блокнот 100 страниц", "UnitName": "штука", "Price": 10, "Quantity": 4 }
    ]
  }
]
