# Тестовое задание Click Intellect

**Автор:** Ескендир Толеубеков  
**Проект:** Shop – разделение товаров на группы с ограничением по цене

---

## Цель проекта

Реализовать сервис, который разделяет список товаров на группы так, чтобы:

- Общая цена группы товаров **не превышала 200 евро**.
- Общая цена была **как можно ближе к 200 евро**.

---

## Функционал

### Группировка товаров

- Endpoint: `POST /group-products`
- Получает список товаров из базы данных (или заранее загруженный список).
- Разделяет их на группы, соблюдая ограничение по цене.
- Возвращает список групп с товарами.

---

## Формат входных данных

Список товаров (например, из Excel):

| Наименование       | Единица измерения | Цена за единицу, евро | Количество, шт. |
|-------------------|-----------------|---------------------|----------------|
| Ручка             | штука           | 1,5                 | 150            |
| Бумага А4         | упаковка        | 2,6                 | 50             |
| Доска маркерная   | штука           | 32                  | 11             |
| Блокнот 100 страниц | штука         | 10                  | 5              |
